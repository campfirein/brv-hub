# TypeScript Kickstart

Context bundle for bootstrapping TypeScript projects with best practices.

## What's included

- tsconfig.json best practices and recommended compiler options
- Standard project structure
- Common utility types
- Recommended tooling (build, lint, format, test)
- Error handling patterns

## Usage

Feed `context.md` to your AI agent when starting a new TypeScript project. It provides the context needed to set up a well-structured project from the start.
