# OpenClaw Installation Troubleshooting

This bundle provides a comprehensive troubleshooting guide for the most common failures encountered when installing and configuring OpenClaw — the AI Gateway/Agent framework. It covers permission errors, version mismatches, port conflicts, Docker issues, and platform-specific problems across Windows, macOS, and Linux.

## When to use this bundle

Feed this bundle to your AI coding agents when:
- A user reports an OpenClaw installation or setup failure.
- Debugging environment issues (Node.js versions, PATH, permissions).
- Resolving Docker networking or port conflict problems with the OpenClaw Gateway.
- Supporting users on different platforms (Windows/PowerShell, macOS, Linux, headless VPS).

## What it covers

- **Environment Issues:** Node.js version requirements, PATH configuration, and EACCES permissions.
- **Platform-Specific Fixes:** Windows/PowerShell, macOS Sharp dependency, Linux build tools.
- **Gateway & Docker:** Port conflicts, Docker networking, and Control UI origin rejection.
- **Configuration:** Missing config files, onboarding wizard failures, and plugin extensions.
- **Diagnostic Commands:** Quick-reference commands for checking OpenClaw status and logs.
